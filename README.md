Aggrerate
=========

Dependencies
------------

* Python packages
  - beautifulsoup4
  - Flask
  - Flask-Login
  - py-bcrypt
  - MySQL-python
* C libraries
  - bcrypt

Install Dependencies
------------

Set up virtualenv on a development machine

```
pip install virtualenv
virtualenv venv
source venv/bin/activate
```

Install requirements

```
pip install -r requirements.txt
```
